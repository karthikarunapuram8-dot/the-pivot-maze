# The Pivot Maze

A red-teamer's walk through an agentic AI architecture, ring by ring. Eight defensive rings, one trust anchor, and why composition is the defining failure mode.

## Read

Live at https://karthikarunapuram8-dot.github.io/the-pivot-maze/

## What it is

A field-notes article. The piece walks an attacker's reasoning through each layer of an agentic platform: policy, identity, trusted content, retrieval corpus, long-term memory, tool chaining, tool definitions, audit, and the governance layer at the center.

Each ring is paired with what stops the attempt at that layer, and the pivot to the next ring when it does.

## What it is not

A vulnerability list. A checklist. A maturity model. The point of the piece is that composition is what breaks agentic systems, not single-control failure.

## Status

Build 2026.05. Iterating based on responses from people doing real red-team work on these platforms.
